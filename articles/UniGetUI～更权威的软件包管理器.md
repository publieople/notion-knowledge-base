## 介绍

该项目的主要目标是为 Windows 10 和 Windows 11 上最常用的 CLI 包管理器（如 [Winget](https://learn.microsoft.com/en-us/windows/package-manager/)、 [Scoop](https://scoop.sh/)、 [Chocolatey](https://chocolatey.org/)、 [Pip](https://pypi.org/)、 [Npm](https://www.npmjs.com/)、 [.NET Tool](https://learn.microsoft.com/en-us/dotnet/core/tools/dotnet-tool-install)和 [PowerShell Gallery](https://www.powershellgallery.com/) ）创建一个直观的 GUI 。

---

## 一、引言：Windows用户的十年之痛

1. 传统软件安装困境：在 Windows 系统中，安装软件常面临诸多问题。浏览器寻宝式搜索软件，易陷入广告丛生的下载站，一不小心就下载到捆绑流氓软件的安装包；去官网下载又担心会遭遇“鉴黄”式的甄别，耗费大量时间和精力。手动下载安装包后，还需一步步点击“下一步”，效率低下且容易出错。
2. 官方工具局限性：微软商店中软件资源相对冷清，热门软件缺失，更新不及时，难以满足用户多样化需求。而 WinGet 命令行工具，因其需要记忆复杂的命令参数，对于普通用户来说使用门槛较高，使得很多用户望而却步。
3. 解决方案登场：UniGetUI 应运而生，这款更权威的软件包管理器，将为 Windows 用户终结软件管理乱象，带来全新的便捷体验。

## 二、核心功能解析

1. 多平台整合能力：UniGetUI 支持 8+主流包管理器，包括 Winget、Scoop、Chocolatey、Pip、Npm、.NET 等，能整合各包管理器资源，覆盖 14000+开源软件资源检索，为用户提供了丰富的软件选择。
2. 智能管理中枢：具备跨平台更新检测功能，不仅能检测通过自身安装的软件更新，还能检测微软商店应用的更新情况。同时，拥有系统级卸载能力，即使是 Windows 内置应用，也能轻松卸载，帮助用户保持系统的整洁与高效。
3. 用户体验优化：UniGetUI 能记忆用户的安装偏好，如架构、路径、版本等信息，下次安装相同软件时可自动保存，无需重复设置。还具备后台静默更新模式，用户可告别强制弹窗，避免被打扰，让软件更新在后台悄然进行。

## 三、使用全指南

1. 安装配置：用户可从官网<>正式渠道获取 UniGetUI，按照官网指引进行安装，安装过程中可切换为中文界面。
2. 实战演示：以安装 VLC 播放器为例，在 UniGetUI 主界面搜索栏中输入“VLC”，系统会实时搜索匹配并展示相关软件包列表。选择合适的版本及来源后，点击安装，即可在安装过程中设置安装路径等参数。若要进行批量更新，可在软件更新列表中全选或指定选择需要更新的软件版本，然后点击更新按钮，即可一次性完成多个软件的更新操作。
3. 高阶功能：对于 Scoop 存储桶管理，用户可通过 UniGetUI 方便地添加、更新或移除 Scoop 存储桶，以获取更多软件资源。同时，还支持命令行参数深度应用，通过“unigetui://”协议调用，可实现更灵活的软件管理操作。

## 四、横向对比评测

1. 与传统命令行工具对比：相较于 WinGet 等命令行工具，UniGetUI 无需用户记忆复杂的命令参数，通过直观的图形界面即可完成软件的搜索、安装、更新和卸载等操作，大大降低了使用门槛，提高了操作效率。
2. 与第三方软件商店对比：在纯净度方面，UniGetUI 专注于开源软件资源，无捆绑广告等不良内容，为用户提供出色的纯净体验。其资源覆盖量也十分广泛，涵盖了众多主流软件和实用工具，能满足不同用户的需求。
3. 与同类 GUI 管理器对比：UniGetUI 在资源占用方面表现出色，运行流畅，不会对系统性能造成明显负担。在响应速度上也具有优势，无论是搜索软件还是执行安装、更新等操作，都能快速响应，为用户节省时间。
